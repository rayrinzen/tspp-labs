# 1.1 Бізнес-вимоги (Фізіологічна потреба: Вода)

## Об'єкт дослідження
Процес контролю споживання питної води для підтримки життєдіяльності організму.

## DIKW Модель
* **Data (Дані):** Кількість випитої води (мл), час прийому, поточна вага користувача.
* **Information (Інформація):** Розрахунок дефіциту води до денної норми.
* **Knowledge (Знання):** Розуміння, що при фізичних навантаженнях норма споживання зростає на 20%.
* **Wisdom (Мудрість):** Побудова довгострокової звички пиття для покращення стану шкіри та обміну речовин.

## Vision (Бачення)
**Для** людей, що піклуються про своє здоров’я, **які** часто забувають пити достатньо води, **продукт** "AquaTracker" **є** мобільним додатком, **що** автоматизує облік гідратації, **на відміну від** звичайних нагадувань у календарі.
