# 1.2 Вимоги користувача

## Use Case Діаграма

![Діаграма прецедентів](diagram.png)

> **Примітка:** На діаграмі зображено Актора "Користувач", який пов'язаний лініями з діями: "Внести об'єм води", "Налаштувати профіль", "Отримати звіт".

## Специфікація сценарію (Gherkin)
**Фіча:** Додавання випитої води.
**Сценарій:** Успішне внесення даних про спожиту рідину.
* **Given:** Я відкрив головний екран додатка.
* **When:** Я обираю об'єм "250 мл" та натискаю "Додати".
* **Then:** Система оновлює відсоток виконання денної норми.
* **And:** Виводиться повідомлення "Чудова робота!".
