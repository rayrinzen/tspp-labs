# Тестові сценарії верифікації вимог

## 1. Функціональні вимоги (Black Box)
**Сценарій:** Додавання води.
- **Крок:** Викликати `WaterRecord(500, user1)`.
- **Очікуваний результат:** Створено об'єкт, об'єм = 500, прив'язаний до користувача.

## 2. Нефункціональні вимоги (Валідація)
**Сценарій:** Перевірка на "дурня" (від'ємні значення).
- **Крок:** Спроба ввести `-100` мл води.
- **Очікуваний результат:** Система видає помилку (Error/Exception) і не створює запис.
